## Reflexivity discount (mandatory for any name he touched)

His posts move small caps 10-20% same-day (he says so himself: "people are
starting to think I'm the catalyst"). Post-2026Q1 track record is partially
self-fulfilling; you would be buying his flywheel, not his alpha.

Rules:
1. Record days-since-his-first-post for the name. Same-week entries after a
   viral post = chasing; default `避免追高`.
2. Classify the post type (new thesis / reaffirmation / supplier map / victory
   lap) per companion-skill workflow (d). Only "new bottleneck thesis" with
   external evidence justifies prompt action; validation window is typically
   5-60 trading days, so patience beats speed.
3. Check whether institutions already validated (13F/13G filings, Nordic short
   registers, block trades). Institutions-in = the mispricing has compressed.
4. Names he never covered but that pass the checklist deserve MORE weight than
   echoes of his picks.
