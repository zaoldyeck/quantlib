## 為什麼用質性判斷而非量化篩

量化指標（5y CAGR、ROA、GM）只告訴你「**過去 5 年誰贏了**」。找 alpha 股的本質是 forward-looking——「**未來 5 年誰會繼續贏**」。

舉例：
- 2010 年看 2330：5 年 CAGR 中等、毛利率 40%（不是當時最亮眼），但護城河 thesis 明確（先進製程領先 Samsung 1 代 + Apple 訂單鎖定）→ 量化指標不會告訴你「該重壓 2330」，**質性判斷會**
- 2020 年看 2308：5 年 CAGR 平庸（電源管理是傳產），但 thesis（AI server 電源密度暴增、台達電是 OCP 規格制定者之一）→ 量化看不出來，**質性看得出來**

質性判斷強迫回答：**「這家公司未來 5 年為何仍會 dominant？」**——這個問題的答案才是 alpha 來源。

## 護城河類型

| 類型 | 含意 | 範例 |
|---|---|---|
| **規模 + 技術獨佔** | Capex / R&D 門檻極高 + 技術領先 | 2330（fab + 製程）、2308（電源 + 規格制定）|
| **客戶鎖定 / 高轉換成本** | 切換成本極高（重新認證、re-qualify）| Apple 鏈大立光；NVIDIA 鏈 ABF 三雄 |
| **網絡效應 / 通路** | 多參與者間互相強化 | 統一超商 |
| **品牌 + 無形資產** | 消費者認知、信任 | 巨大捷安特、統一茶裡王 |
| **法規 / 特許** | 政策保護、執照稀缺 | 中華電固網、金控業 |
| **利基專業化** | 小市場做到全球前幾大 | 5274 信驊 BMC、3529 力旺 IP、8069 元太電子紙 |

### P.4 量化 vs 質性的權衡規則（v12 新增）

1. **量化過去、質性未來** — 5y CAGR 是 lagging 結果，forward thesis 是 leading 預測；兩者方向不同時，**質性 Tier > 量化排名**
2. **量化做 sanity check** — 質性升 Tier 2A 但 ROA < 5% / GM < 15% → 必須 hedge（如 3037 欣興案例）
3. **量化作為 sunset 確認** — 5y CAGR < 5% 且質性 thesis 也弱 → 直接 sunset（如 3034 聯詠）
4. **量化高但未列名** — 必須補充 thesis WebSearch（如 3529 力旺，可能漏掉的 emerging leader）
