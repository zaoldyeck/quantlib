## 月結協定

每月營收 refresh(~11 日)後的第一個週末:

1. **實現側**:live book 月內 NAV 變化(lot 錨計)、出入場清單與觸發門、實際成交價。
2. **模擬側**:引擎同期模擬 NAV(`engine.py --start 2025-01-01`,取當月段)。
3. **差異歸因**:fill 差(UMEE 執行 vs 收盤價假設)、時點差(T+1 執行 vs 訊號日)、
   人工偏離(未送單/否決的建議)。
4. 追加一行到下表,git commit。
